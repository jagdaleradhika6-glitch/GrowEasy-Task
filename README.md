**GrowEasy CRM Import Wizard**

An AI-powered CSV Import Wizard built with Next.js, TypeScript, and Gemini AI for seamless CRM contact imports. The application intelligently maps CSV columns to CRM fields, validates data, previews records, and imports contacts with real-time progress tracking.

**🚀 Features**

- CSV File Upload
- AI-powered Column Mapping (Gemini API)
- Data Preview Before Import
- Data Validation
- Import Progress Tracking
- Batch Processing
- Responsive UI
- Error Handling with Zod Validation

**🛠️ Tech Stack**

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- Gemini AI
- Zod
- Framer Motion
